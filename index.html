NS Cloud Infrastructure with IDS

Cloud infrastructure on Proxmox with VMs, OpenWrt router, and SIEM/SOAR Control Box (Wazuh + Suricata) for Intrusion detection. VMs have unique SSH keys and Wazuh agents. Router manages VLANs, subnets, and firewall. Control Box detects threats and sends alerts via e-mail. Admin accesses via tunnel (Twingate), users via SSH on browser with restricted routing. Also a reported cyber attack simulation for testing the alerts.
