Projects

Pakisrv Cloud Infrastructure

Cloud infrastructure on Proxmox with VMs, OpenWrt router, and Control Box (Wazuh) for Intrusion detection. VMs have unique SSH keys and Wazuh agents. Router manages VLANs, subnets, and firewall. Control Box detects threats and sends alerts. Admin accesses via tunnel (Twingate), users via SSH with restricted routing. Attacks: (1) Lateral movement via weak SSH, detected by Control Box; (2) DDoS (TCP flood) on the router, mitigated by traffic monitoring and IP blocking.
